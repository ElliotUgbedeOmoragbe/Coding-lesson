Webdesign

Webdesign is the process of designing and implementing a website

What do you need to know

  • HTML(Hyper Text Markup Language):

    HTML is the building block of the web it gives skeleton to our webpage. when creating a web file we name the file extension of the text document (.html)

  • CSS(Cascading Style Sheets):

    CSS is used for beautifying our page it add flesh to the skeloton i.e html. We create a style sheets by adding a file extension of (.css) to a text document.

  • Javascript:

    Javascript is used to give brain(logic) to our flesh and bone (html,and css) .We create ajavascript file by naming a text document file extension (.js)
